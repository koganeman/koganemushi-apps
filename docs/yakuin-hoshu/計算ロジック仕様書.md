## 3. VBA関数の定義

### 3.1 KoujyogoTS - 給与所得控除後の金額（通常）

**令和7年改正対応**

引数: `Num` = 給与収入（円）

```
if Num < 650,000:
    SNum = 0
elif Num <= 1,900,000:
    SNum = Num - 650,000
elif Num < 3,600,000:
    SNum = Num × 0.7 - 80,000
elif Num < 6,600,000:
    SNum = Num × 0.8 - 440,000
elif Num < 8,500,000:
    SNum = Num × 0.9 - 1,100,000
else:
    SNum = Num - 1,950,000
```

### 3.2 Koujyogo_ch - 給与所得控除後の金額（子育て介護世帯）

**令和7年改正対応** - 850万円超の上限が1,000万円に引き上げ

引数: `Num` = 給与収入（円）

```
if Num < 650,000:
    SNum = 0
elif Num <= 1,900,000:
    SNum = Num - 650,000
elif Num < 3,600,000:
    SNum = Num × 0.7 - 80,000
elif Num < 6,600,000:
    SNum = Num × 0.8 - 440,000
elif Num < 10,000,000:          ← 通常は8,500,000
    SNum = Num × 0.9 - 1,100,000
else:
    SNum = Num - 2,100,000      ← 通常は1,950,000
```

### 3.3 給与所得金額の計算（セル数式）

セルD14の配列数式:
```
= IF(D84=FALSE, KoujyogoTS(D8+D9+D10+D11+D12-D13), Koujyogo_ch(D8+D9+D10+D11+D12-D13))
```

**計算内容**: 給与収入 = 定期同額 + 事前確定1〜3回 + 他の給与収入 - 確定給付年金掛金

子育て介護世帯フラグ(D84)がTRUEの場合は `Koujyogo_ch`、FALSEの場合は `KoujyogoTS` を使用。

### 3.7 SocialInsBTS - 賞与の社会保険料（健康保険のみ）

引数: `Income` = 賞与額（円）, `Age` = 年齢

**計算**:
1. `Income = Int(Income / 1000) * 1000` （千円未満切り捨て = 標準賞与額）
2. 上限チェック: `Hmax`（健保・介護標準賞与額上限 O11）を超える場合は上限で計算

```
年齢0〜39, 65〜74:
    H = min(Income, Hmax) × Hins / 2

年齢40〜64:
    H = min(Income, Hmax) × (Hins + Cins) / 2
```

※ この関数は健康保険部分のみを返す（厚生年金部分は含まない）

---

### 3.8 SocialInsBPTS - 賞与の厚生年金保険料

引数: `Income` = 賞与額（円）, `Age` = 年齢

**計算**:
1. `Income = Int(Income / 1000) * 1000` （千円未満切り捨て = 標準賞与額）
2. 上限チェック: `Pmax`（厚生年金標準賞与額上限 O12 = 1回150万円）を超える場合は上限で計算

```
年齢0〜65:
    P = min(Income, Pmax) × Pins / 2
```

### 4.2 課税所得金額

**セルD20**:
```
= INT(
    IF(
      (給与所得金額 + 配当所得 + 他の所得金額 - 社会保険料控除額 - 他の所得控除額 - 基礎控除額) > 0,
      (給与所得金額 + 配当所得 + 他の所得金額 - 社会保険料控除額 - 他の所得控除額 - 基礎控除額),
      0
    ) / 1000
  ) * 1000
```

→ 1,000円未満切り捨て。マイナスの場合は0。

### 4.4 配当控除額（所得税）

**セルD22 → D99を参照**

D99の計算ロジック:
```
前所得(D102) = 課税所得金額(D20) - 配当所得(D15)
配当(D103) = 配当所得(D15)

IF (前所得 + 配当) <= 10,000,000:
    配当控除 = 配当 × 0.10
ELIF 前所得 >= 10,000,000:
    配当控除 = 配当 × 0.05
ELIF 配当 <= (10,000,000 - 前所得):
    配当控除 = 配当 × 0.10
ELSE:
    配当控除 = (10,000,000 - 前所得) × 0.10 + (配当 - (10,000,000 - 前所得)) × 0.05
```

→ 課税所得1,000万円以下部分は10%、超部分は5%で控除

---

### 4.5 住民税

**セルD24**: `= 課税所得金額(D20) × 0.1`

### 4.6 配当控除額（住民税）

**セルD25 → D100を参照**

D100の計算ロジック（D99と同じ構造、料率のみ異なる）:
```
IF (前所得 + 配当) <= 10,000,000:
    配当控除 = 配当 × 0.028
ELIF 前所得 >= 10,000,000:
    配当控除 = 配当 × 0.014
ELIF 配当 <= (10,000,000 - 前所得):
    配当控除 = 配当 × 0.028
ELSE:
    配当控除 = (10,000,000 - 前所得) × 0.028 + (配当 - (10,000,000 - 前所得)) × 0.014
```

→ 課税所得1,000万円以下部分は2.8%、超部分は1.4%で控除

### 7.2 ctax関数 - 法人税計算

引数: `Num` = 法人所得金額（繰越欠損金控除後）

実効税率 a, b, c を法人税シートのG6, G7, G8から取得。

```
IF Num <= 4,000,000:
    zei = Num × a
ELIF Num <= 8,000,000:
    zei = Num × b - 4,000,000 × (b - a)
ELIF Num > 8,000,000:
    zei = Num × c - 4,000,000 × (c - a) - 4,000,000 × (c - b)
```

### 7.3 法人税の計算フロー

**現状** (C5):
```
法人所得 = 役員報酬支払前法人所得(P3) - 役員報酬合計(N8+N9+N10+N11) - 会社負担社保合計(N32)
```

**法人税** (C8):
```
= IF(法人所得 - 繰越欠損金 <= 0, 均等割, ctax(法人所得 - 繰越欠損金) + 均等割)
```

## 10. 計算の全体フロー

```
入力: 定期同額、事前確定給与、配当所得、他所得、控除額、年齢、料率等
  ↓
1. 給与収入の計算
   給与収入 = 定期同額 + 事前確定1〜3 + 他の給与収入 - 確定給付年金掛金
  ↓
2. 給与所得金額の計算 (KoujyogoTS / Koujyogo_ch)
  ↓
3. 社会保険料の計算
   a. 定期同額ベース: health(), pension() で月額→年額
   b. 事前確定ベース: SocialInsBTS(), SocialInsBPTS() で賞与保険料
   c. 社会保険料計 = 健康保険料 + 厚生年金保険料
  ↓
4. 所得控除の計算
   a. 社会保険料控除額 = 社会保険料計（個人負担分）
   b. 基礎控除額 = VLOOKUPテーブルから取得
  ↓
5. 課税所得金額の計算
   = (給与所得 + 配当所得 + 他の所得 - 社保控除 - 他の所得控除 - 基礎控除)
   → 1,000円未満切り捨て、マイナスは0
  ↓
6. 税金の計算
   a. 所得税 = incometaxTS(課税所得) × 1.021
   b. 配当控除額（所得税）: 1,000万円境界で10%/5%
   c. 住民税 = 課税所得 × 10%
   d. 配当控除額（住民税）: 1,000万円境界で2.8%/1.4%
   e. 個人税金合計 = (所得税 - 配当控除所得税 - 税額控除) + (住民税 - 配当控除住民税)
  ↓
7. 手取り額の計算
   = 給与収入合計 + 配当所得 + 他の所得 - 税金＋社会保険料
  ↓
8. 会社負担の計算
   = health + pension + 子育て拠出金 + 事前確定社保
  ↓
9. 法人税の計算
   法人所得 = 法人所得(税引前) - 役員報酬合計 - 会社負担社保合計
   法人税 = ctax(法人所得 - 繰越欠損金) + 均等割
```
